# FutMon World

RPG web de exploración para móvil. Recorre un mundo pixelado, encuentra futbolistas en las zonas de entrenamiento, enfréntate a ellos por turnos y fíchalos con balones contrato.

## Jugar

La versión publicada está disponible en GitHub Pages.

## Desarrollo local

No necesita instalación ni servidor:

1. Descarga el repositorio.
2. Abre `index.html` en el navegador.

El progreso, equipo, posición y balones se guardan localmente mediante `localStorage`.

## Publicación

Cada cambio enviado a `main` se publica automáticamente con GitHub Actions.

Este es un proyecto fan no oficial. No está afiliado ni respaldado por clubes, ligas o futbolistas. No utiliza fotografías ni escudos oficiales.
