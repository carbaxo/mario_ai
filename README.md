# FutMon World

RPG web de fútbol tipo *monster-catcher*, en español y sin dependencias. Recorre un mundo pixelado de 48×48 casillas con cuatro regiones, encuentra futbolistas en la hierba alta, debilítalos en duelos por turnos y fíchalos con balones contrato.

## Características

- 🗺️ Mundo panorámico con cámara y movimiento suaves, minimapa, partículas y ambiente distinto por región (Villa Cantera, Ciudad Solar, Bosque Cantera y Distrito Nocturno).
- ⚔️ Duelos por turnos claros: daño visible en cada botón, números de daño flotantes, indicador de turno, ventaja táctica por posición (DEL > MED > DEF > DEL, el portero resiste disparos) y % de fichaje en vivo.
- 📖 Historia de 10 capítulos con jefe final: el Entrenador Sombra y la Copa de las Leyendas.
- 🧑‍🏫 Cinco entrenadores jugables, cada uno con una habilidad única de combate.
- 🪙 Economía: monedas por victoria, tiendas, cofres con recompensas, bebidas isotónicas y centro médico.
- 🔊 Efectos de sonido sintetizados con WebAudio (silenciables) y tutorial integrado en el primer duelo.

## Jugar

La versión publicada está disponible en GitHub Pages.

## Desarrollo local

No necesita instalación ni servidor:

1. Descarga el repositorio.
2. Abre `index.html` en el navegador.

El progreso, equipo, posición y objetos se guardan localmente mediante `localStorage`.

## Publicación

Cada cambio enviado a `main` se publica automáticamente con GitHub Actions.

Este es un proyecto fan no oficial. No está afiliado ni respaldado por clubes, ligas o futbolistas. No utiliza fotografías ni escudos oficiales.
